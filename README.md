# fortran-load 

Server Program for Making Electricity Load Database

- Collects electricy load data hourly from TEPCO "Denki Yoho".

# Dependencies

- fortran-logger
- fortran-datetime
- fortran-sqlite3

# Install

> cd install\
> sudo make service

# Uninstall

> cd install\
> sudo make stop\
> sudo make uninstall
